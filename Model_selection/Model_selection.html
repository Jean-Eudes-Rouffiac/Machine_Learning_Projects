1 Classification de données Splice¶

 

 

Les donnes Splice sont des donnes de squences d'ADN auxquelles l'ADN superflux est retir durant le processus de cration de protines dans les organismes. Le problme que nous voulons rsoudre est de reconnatre, pour une squence d'ADN donne, les frontires entre les exons et les introns.

Chargeons dans un premier temps les donnes.

 

In [27]:
from sklearn.datasets import load_svmlight_file
import numpy as np

X, Y = load_svmlight_file("./splice")

 

On veut maintenant apprendre un modle SVM non-linaire avec un noyau RBF (gaussien) de paramtre $\gamma$ : $k(x, z) = exp(\gamma \lVert x z \rVert^2)$ en slectionnant de faon optimal les hyper paramtres C du SVM et donc $\gamma$ du noyau. On se servira du module GridSearchCV.

Dans un premier temps, on construit une chelle logarithmique les deux vecteurs suivants qui vont tre les valeurs possibles que peuvent prendre C et $\gamma$.

 

In [10]:
gamma_grid = np.logspace(-3, 0, 30)
C_grid = np.logspace(-0.5, 1.5, 30)

 

On effectue alors une Grid Search sur la grille forme par les vecteurs prcdemment dfinis.

 

In [11]:
from sklearn.model_selection import GridSearchCV
from sklearn.svm import SVC
# the grid
parameters = [{"gamma": gamma_grid, "C": C_grid}]
# the classifier
clf_rbf = SVC(kernel="rbf", tol=0.01)
# Perf a K-fold validation using the accuracy as the performance measure
K = 10 # feel free to adapt the value of $K$
# we will dot it on a grid search using n_jobs processors
clf_rbf = GridSearchCV(clf_rbf, param_grid=parameters, cv=K, scoring="accuracy", verbose=1, n_jobs = 2)

 

Lanons la slection du modle optimal et son apprentissage
